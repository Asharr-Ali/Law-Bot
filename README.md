# LawBot – AI-Based Legal Assistant

LawBot is an AI-powered legal assistant that uses Retrieval-Augmented Generation (RAG) to provide context-aware and grounded legal responses. Instead of relying solely on generative models, LawBot retrieves relevant legal information from a structured dataset before generating answers, improving accuracy and reliability.

---

## Overview

Large language models often produce vague or hallucinated responses in sensitive domains like law. LawBot addresses this by combining structured legal data with semantic retrieval techniques.

The system emphasizes a data-centric approach:
- Clean and structured legal dataset
- Embedding-based retrieval
- Context-grounded response generation

---

## Features

- Retrieval-Augmented Generation (RAG)
- Custom structured legal dataset (JSON)
- Semantic search using embeddings
- Context-aware response generation
- Reduced hallucination through SAC


---

## Tech Stack

- Frontend : Next js
- Backend: FAST API 
- LLM Integration: Groq API  
- Vector Database: ChromaDB  
- Data Format: JSON  
- Core Concept: Retrieval-Augmented Generation (RAG)
